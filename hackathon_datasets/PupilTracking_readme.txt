These file contain the first 10min of pupil tracking for an animal;

video file contains:
    18_000 frames @ 30Hz

pickle file contains behavioral data:
    'time' : time since the beginning of the experiment
    'recording_duration': time from the beginning of the 2p acquisition
    'video_frame_idx': id of the pupil video currently acquired 
    'position': position of the animal in a virtual track (0 - 500cm)
    'velocity': velocity of the animal in a virtual track
